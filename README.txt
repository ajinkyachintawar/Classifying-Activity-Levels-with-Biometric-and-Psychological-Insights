===========================================
README: Running the Code in JupyterLab/Notebook
===========================================

1. Prerequisites:
   - Ensure Python 3.9.18 is installed.
   - JupyterLab Version: 4.0.11 or Jupyter Notebook	
   - Required libraries:
     Install the following libraries in a single cell using `pip`:
     ```
     pip install pandas==1.4.2
     pip install plotly==5.6.0
     pip install seaborn==0.11.2
     pip install matplotlib==3.5.1
     pip install scikit-learn==1.0.2
     pip install imbalanced-learn==0.12.3
     pip install joblib==1.4.2
     ```

2. Running the Code:
   - Open the Anaconda Command Prompt and navigate to the project folder:
     ```
     cd /path/to/project/
     ```
   - Launch JupyterLab:
     ```
     jupyter lab
     ```
   - Open and run `Classification_ActivityLevel.ipynb`:
     - Use "Restart Kernel and Run All Cells."
   - Open and run `Deployment.ipynb`:
     - Use "Restart Kernel and Run All Cells."

3. Tableau Dashboard:
   - Refer to Section 6.4 of the report for the Tableau dashboard link to view interactive results.
